             ПОДГОТОВКА К РАБОТЕ СО СБОРКОЙ GULP ОТ DATOSH CODE
             

1. Распаковать сборку Gulp в пустую папку.

Если на компьютере не установлен Node.js и Gulp, то:
1. Установить Node.js: https://nodejs.org/ru/ 

2. Открыть терминал в корне проекта и установить Gulp на компьютер командой:
 npm i gulp -j
------------------------------------------------------------------------

2. Открыть терминал в корне проекта и установить Gulp в проект: 
 npm i gulp --save-dev

3. Устанавливаем Sass командой:
 npm install sass gulp-sass --save-dev

4. Проверяем работоспособность сборки. В терминале вводим команду: gulp
Если отобразится окно приветствия, как указано на скриншоте, то сборка рабочая.

5. Работаем в сборке. Где код html пишем в index.html, scss складываем в папку /scss. 
Изображения, скрипты и шрифты складываем в папку /app.

Для отображения результата своей работы, в терминале вводим команду: gulp
Для выхода, в терминале нажимаем комбинацию: Ctrl+C.

Для сборки готового проекта, следует ввести команду: gulp build 
После чего появится папка dist, содержимое которой можно выкладывать на хостинг.
